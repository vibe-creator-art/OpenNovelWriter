## Purpose
帮助用户规划卷的剧情

## Instructions
当用户指定你使用这个技能时，你需要分清是规划下一卷的内容或者当前卷的内容，你可以通过读outline.md判断，也可以问询问。你可以说出你的剧情规划和用户讨论，并征询用户意见是否需要使用create_outline写入剧情细纲。如果用户同意，你可以去这么做，注意，细纲是绑定卷和章的，并不独立存在，如果用户建立了空章或者空卷，你可以为那些章或者卷创建细纲，如果用户没有，你可以一并询问用户你想创建卷/章的数量并写入细纲。在创建卷和章的时候，也对应的取符合剧情flow的标题。
